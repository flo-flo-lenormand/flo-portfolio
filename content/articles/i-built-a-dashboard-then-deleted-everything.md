---
title: "I built a dashboard with AI agents, then deleted everything"
date: "2026-03-16"
description: "The paradox of building with AI - you can build so fast that you outrun your actual needs."
published: false
---

I spent weeks building a personal productivity dashboard with Claude Code. It had everything:

- A background agent (Stuart) that scanned my Google Chat every 5 minutes, drafted replies, and queued them for my approval
- A morning briefing system that summarized overnight activity
- A calendar integration showing my day
- A "What's Happening" section aggregating project updates
- A creative assistant (Tom) scraping design inspiration from Mobbin
- A content agent (Mel) drafting Threads posts
- A third agent (Jerry) that never got past the placeholder stage
- Draft reply UI with send/edit/dismiss controls inline on each task
- Quick capture input for adding tasks manually
- Project cards with collapsible details

It looked impressive. Multiple agent pages in the sidebar, each with their own personality and avatar. Animated breathing dots. Mood indicators. The works.

Then I started actually using it.

The draft replies sat there unreviewed because I'd already replied in GChat directly. The calendar section told me things I already knew from looking at my phone. The morning briefing was interesting but not actionable. The inspiration grid was cool but I never opened it. Jerry's page was literally empty.

So I deleted it. All of it.
